Google Maps API V3

Placer un marqueur sur la carte

On utilise un objet de type google.maps.Marker avec des paramètres placés dans un tableau statique.

  • position : objet de type LatLng qui définit la position du marqueur
  • map : objet de type Map qui définit la carte sur laquelle positionner le marqueur
  • title : le texte affiché au survol
  • icon : objet de type MarkerImage qui définit l'image à afficher
